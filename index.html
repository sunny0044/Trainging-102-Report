Projects

  • The G Connects

    TheG Connect Hub is a web application for an authenticated user to list, display, search, and manage business profiles. It stores profile data using the JSONBin API and includes features like simulated 'Meta AI' promotion and collaboration requests for each listing. Features

    1. Secure Authentication System: Implements a simple login/logout mechanism with a hardcoded username and password (admin/1234) to control access to the main dashboard.
    2. Dynamic Business Profile Management: Allows users to add new business listings by submitting details (name, contact, Instagram, etc.) through a dedicated form.
    3. Real-Time Data Persistence: Utilizes the JSONBin API to store, retrieve, and update the list of business profiles, ensuring data is saved across sessions.
    4. Interactive Profile Search: Features a live search bar that filters the displayed business profiles instantly based on keywords in the business name or description.
    5. Simulated Engagement Actions: Provides interactive buttons on each profile card to simulate "Meta AI" promotion (randomly increasing views) and initiate collaboration requests with a simple alert.

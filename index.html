  • AudioCrypt Project

    This project is a secure audio steganography tool developed to demonstrate how encrypted messages can be hidden within audio files using cryptographic and steganographic techniques. It uses AES-256 encryption and LSB-based data hiding methods, all wrapped in a GUI application built with Python and Tkinter.

    Features
    1. Supports embedding secret messages into .wav audio files using Least Significant Bit (LSB) steganography.
    2. Encrypts messages before hiding them using AES-256 (CBC mode) for added security.
    3. Allows password protection for decrypting hidden messages securely.
    4. Provides buttons to select audio files, input messages, encrypt & embed, or extract & decrypt.
